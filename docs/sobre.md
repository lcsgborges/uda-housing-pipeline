## Projeto

Housing Data Intelligence é um pipeline UDA para estruturar dados habitacionais a partir de documentos não estruturados, com API FastAPI, extração por LLM, PostgreSQL, RustFS, documentação MkDocs Material e CI/CD no GitHub Actions.
